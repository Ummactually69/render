📋 Hướng dẫn nhập JSON nhanh

Dán JSON theo định dạng bên dưới vào ô trên, nhấn Nhập JSON:

[
  {
    "q": "Câu hỏi của bạn?",
    "img": "https://url-hinh-anh.jpg",
    "options": ["Đáp án A", "Đáp án B", "Đáp án C", "Đáp án D"],
    "correct": 0
  }
]

correct: chỉ số đáp án đúng (0=A, 1=B, 2=C, 3=D) hoặc "ALWAYS_TRUE".
